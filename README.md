# 🎯 SmartTailor — AI Resume × Job Description Matcher

SmartTailor is an AI-powered web application that analyzes your resume against any job description, providing a match score, identifying skill gaps, and rewriting your experience to improve ATS compatibility.

---

## 🚀 Live Demo

👉 **[Try SmartTailor Now](https://smarttailor-resume-checker-ob3vrdy4eqyf2ukjye43nc.streamlit.app/)**

> 🔗 Upload your resume and test it against real job descriptions for best results.

---

## 🌟 Key Features

- 📄 **Resume Upload & Parsing**  
  Upload PDF/TXT resumes or paste plain text for instant processing.

- 🌐 **Job Description Scraper**  
  Paste any job URL — the app extracts and analyzes the description automatically.

- 📊 **Match Score (0–100%)**  
  Get a realistic compatibility score with a concise explanation.

- 🧠 **Skill Gap Analysis**  
  Identifies missing technical skills and keywords required for the role.

- ✍️ **AI Bullet Rewriter**  
  Converts generic resume points into **STAR-formatted, ATS-optimized achievements**.

## 🛠️ Tech Stack

| Category              | Technologies Used |
|----------------------|------------------|
| Frontend             | Streamlit (Custom Glassmorphism UI) |
| AI / LLM             | LangChain + Groq (LLaMA 3.1 8B Instant) |
| Document Parsing     | PyPDF2 |
| Web Scraping         | BeautifulSoup4, Requests |
| Environment Handling | Python-dotenv |

---

## 💡 How It Works

1. Upload or paste your resume  
2. Provide a job description (URL or text)  
3. AI analyzes compatibility  
4. Receive:
   - Match score  
   - Missing skills  
   - Optimized resume bullets
